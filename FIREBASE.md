# Firebase Setup

This project uses a shared Firebase project for authentication.

## Setup

1. Obtain Firebase configuration values from team member with Firebase Console access.
2. Copy the values into your `.env.local` file.
3. Do not enable additional authentication providers unless the team has agreed to do so.

> DO NOT COMMIT `.env.local` to the repository.

## Project Details

**Firebase Project:** Smart Study Planner

**Authentication Providers:**
- Email/Password
